# classifier-models
Data classifiers and exploratory data analysis applied to the Enem 2021 dataset.

*The data classifiers models used were:*

- Naive Bayes;
- KNN;
- Random Forest;
- Logistic Regression;
- Decision Tree

*Tools used to help in the project development:*

- Pandas;
- GeoPandas;
- Numpy;
- Scikit learn;
- Matplotlib
- Seaborn
